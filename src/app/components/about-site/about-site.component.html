¿Cómo hice este sitio?

Planeación

La idea detrás de este sitio es que funcione como portafolio de mi trabajo, demostrar de forma visible las cosas que se hacer, así como un medio para aplicar nuevos conocimientos conforme vaya aprendiendo nuevas y mejores herramientas.

Antes de comenzar a diseñar el sitio decidí qué las cosas quería mostrar fueron:

  • El mostrar de forma gráfica mis habilidades.
  • Dejar a la mano mi CV y una forma más para contactarme.
  • Mostrar un poco de mi experiencia como ingeniero.
  • Dar a conocer un poco más sobre quién soy.
  • Exponer de forma breve el desarrollo del sitio.
  • Que se viera bien tanto en dispositivos móviles como en pantallas grandes.

Diseño

Para el diseño del sitio utilicé NinjaMock para realizar los mockups del sitio. Me gusta mucho porque te permite diseñar interfaces gráficas de manera rápida y sencilla, además de poder jugar con los tamaños entre dispositivos y simular la navegación del sitio.

Para las fuentes opté por Google Fonts ya que ofrecen una amplia selección de fuentes de uso libre para usar en el sitio. En los títulos se utiliza la fuente Serif llamada Merriweather, para textos y demás se usa la fuente Sans Serif llamada Oswald. Las escogí porque me gustó como se ve mi nombre escrito en ellas.

Para escoger la paleta de colores primero busqué tonos azules en la web de Pantone después de seleccionar uno que me gustó usé el sitio de Paletton para descubrir los tonos adyacentes y los complementarios.

Desarrollo

Actualmente el sitio se encuentra desarrollado en Angular y está alojado en un repositorio en Github. Para apoyarme con los estilos utilicé Bootstrap, además de FontAwesome para el manejo de íconos. Decidí usar estas herramientas porque tengo bastante familiaridad con ellas.

Se siguieron principios de diseño web como responsive design y mobile-first, así como nociones de usabilidad para evitar dar una mala experiencia de usuario. Se implementó Google Analytics para observar y analizar el progreso del sitio. Provicionalmente la información recolectada en los formularios de contacto se almacena en una base de datos en Firebase.

Despliegue

Como se puede observar, este sitio se encuentra hosteado en Github Pages. Siendo honesto me gustó mucho lo sencillo que es de utilizar. Espero en un futuro cercano adquirir un dominio personal en el cual alojar la aplicación para poder integrar más funcionalidades.
